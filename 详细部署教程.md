# 🎬 电影收藏网站详细部署教程

## 第一步：准备GitHub仓库

### 1.1 注册GitHub账户（如果还没有）
1. 访问 [github.com](https://github.com)
2. 点击右上角"Sign up"注册新账户
3. 填写用户名、邮箱、密码等信息
4. 完成邮箱验证

### 1.2 创建新仓库
1. 登录GitHub后，点击右上角"+"号
2. 选择"New repository"
3. 填写仓库信息：
   - **Repository name**: `cinema-shelf`（或您喜欢的名字）
   - **Description**: "我的电影收藏网站"
   - **Public**（选择公开，这样部署是免费的）
   - **Initialize this repository with a README**（勾选）
4. 点击"Create repository"

### 1.3 上传项目文件
有两种方法上传文件：

#### 方法一：使用GitHub网页上传（推荐新手）
1. 进入您刚创建的仓库页面
2. 点击"Add file" → "Upload files"
3. 将您桌面上的`Kanye`文件夹中的所有文件拖拽到上传区域
4. 确保上传以下文件：
   - `index.html`
   - `app.js`
   - `styles.css`
   - `vercel.json`
   - `api/index.js`
   - `assets/`文件夹中的所有文件
   - `DEPLOYMENT_GUIDE.md`
5. 在下方填写提交信息："Initial commit: 电影收藏网站"
6. 点击"Commit changes"

#### 方法二：使用Git命令（如果熟悉Git）
```bash
# 克隆仓库到本地
git clone https://github.com/您的用户名/cinema-shelf.git

# 复制项目文件到仓库文件夹
cp -r Kanye/* cinema-shelf/

# 添加所有文件
git add .

# 提交更改
git commit -m "Initial commit: 电影收藏网站"

# 推送到GitHub
git push origin main
```

## 第二步：部署到Vercel

### 2.1 注册Vercel账户
1. 访问 [vercel.com](https://vercel.com)
2. 点击"Sign up"
3. 选择"Continue with GitHub"（推荐）
4. 授权Vercel访问您的GitHub账户

### 2.2 部署网站
1. 登录Vercel后，点击"New Project"
2. 在"Import Git Repository"中，选择您刚创建的`cinema-shelf`仓库
3. 点击"Import"

### 2.3 配置项目设置
Vercel会自动检测到您的配置，但请确认：
- **Framework Preset**: 应该自动识别为"Other"
- **Root Directory**: 保持默认（./）
- **Build Command**: 留空
- **Output Directory**: 留空
- **Install Command**: 留空

### 2.4 开始部署
1. 点击"Deploy"按钮
2. 等待部署完成（通常需要1-3分钟）
3. 部署成功后，您会看到"Congratulations!"页面
4. 记下您的网站地址，格式为：`https://cinema-shelf.vercel.app`

## 第三步：测试您的网站

### 3.1 访问网站
1. 在浏览器中打开您的网站地址
2. 检查网站是否正常显示
3. 测试基本功能：
   - 页面加载是否正常
   - 按钮是否可以点击
   - 电影收藏功能是否工作

### 3.2 功能测试
1. **添加电影测试**：
   - 点击"添加到收藏"按钮
   - 检查电影是否出现在收藏列表中

2. **标记已观看测试**：
   - 点击"标记已观看"按钮
   - 检查状态是否更新

## 第四步：自定义设置（可选）

### 4.1 自定义域名
如果您有自己的域名：
1. 在Vercel项目设置中点击"Domains"
2. 输入您的域名（如：`movies.yourname.com`）
3. 按照提示配置DNS记录

### 4.2 环境变量配置
如果需要配置API密钥等：
1. 在Vercel项目设置中点击"Environment Variables"
2. 添加需要的环境变量

## 常见问题解决

### ❓ 问题1：部署失败
**症状**：Vercel显示部署失败
**解决方法**：
1. 检查GitHub仓库中的文件是否完整
2. 确认`vercel.json`配置文件正确
3. 查看Vercel的部署日志了解具体错误

### ❓ 问题2：网站显示空白
**症状**：网站能打开但显示空白页面
**解决方法**：
1. 检查浏览器控制台是否有JavaScript错误
2. 确认所有静态文件都已上传
3. 检查API路径配置是否正确

### ❓ 问题3：功能不工作
**症状**：页面显示正常但功能无法使用
**解决方法**：
1. 检查网络连接
2. 确认API服务器是否正常运行
3. 查看浏览器控制台错误信息

### ❓ 问题4：手机显示不正常
**症状**：在手机上显示错乱
**解决方法**：
1. 网站已经做了响应式设计，通常会自动适应
2. 如果仍有问题，检查CSS媒体查询
