## 第一步：创建GitHub仓库（15分钟）

### 1.1 访问GitHub并登录
- 打开浏览器，访问：`https://github.com`
- 如果您已有账户，点击"Sign in"登录
- 如果没有账户，点击"Sign up"注册

### 1.2 注册GitHub账户（如需要）
1. 输入用户名（如：`jiangzhengjie`）
2. 输入邮箱地址
3. 设置密码
4. 完成人机验证
5. 点击"Create account"
6. 检查邮箱，点击验证链接

### 1.3 创建新仓库
1. 登录后，点击页面右上角的"+"号
2. 选择"New repository"
3. 填写仓库信息：
   - **Owner**: 您的用户名（自动填充）
   - **Repository name**: `my-cinema-collection`
   - **Description**: `我的个人电影收藏网站`
   - **Public** ✅（选择公开，免费）
   - **Add a README file** ✅（勾选）
4. 点击绿色的"Create repository"按钮

### 1.4 上传项目文件
1. 进入您刚创建的仓库页面
2. 点击"Add file"按钮
3. 选择"Upload files"
4. 打开文件资源管理器，找到您的`Kanye`文件夹
5. **全选所有文件**（Ctrl+A），然后拖拽到GitHub的上传区域
6. 等待所有文件上传完成
7. 在下方填写：
   - **Commit message**: `上传电影收藏网站文件`
8. 点击"Commit changes"

**✅ 第一步完成！** 您的代码现在已经安全存储在GitHub上。

---

## 第二步：部署到Vercel（10分钟）

### 2.1 访问Vercel并注册
1. 打开新标签页，访问：`https://vercel.com`
2. 点击"Sign up"
3. 选择"Continue with GitHub"（推荐）
4. 授权Vercel访问您的GitHub账户

### 2.2 导入项目
1. 登录后，点击"New Project"
2. 在"Import Git Repository"部分，您会看到刚创建的仓库
3. 点击仓库名称旁边的"Import"按钮

### 2.3 配置项目（重要！）
Vercel会自动检测配置，但请确认：

**项目设置检查清单：**
- [ ] **Framework Preset**: 显示为"Other" ✅
- [ ] **Root Directory**: 保持为`./` ✅
- [ ] **Build Command**: 留空 ✅
- [ ] **Output Directory**: 留空 ✅

### 2.4 开始部署
1. 点击蓝色的"Deploy"按钮
2. 等待部署过程（约1-3分钟）
3. 您会看到部署进度条

### 2.5 获取网站地址
部署完成后：
1. 您会看到"Congratulations!"页面
2. **记下您的网站地址**，格式为：
   `https://my-cinema-collection.vercel.app`
3. 点击地址即可访问您的网站

**✅ 第二步完成！** 您的网站现在已经在线了！

## ❓ 遇到问题？

### 常见问题快速解决：

**问题：部署失败**
- 检查GitHub文件是否完整上传
- 确认`vercel.json`文件存在

**问题：网站显示空白**
- 刷新页面
- 检查浏览器控制台（F12）是否有错误

**问题：功能不工作**
- 确认网络连接正常
- 检查API是否正常响应

如果以上方法无法解决，可以：
1. 重新阅读本指南
2. 查看详细的错误信息
3. 寻求技术朋友帮助
